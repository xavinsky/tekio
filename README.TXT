Tekio is an old publication system in python 2.5.

It's an internal programm, then there is no documentation.

Functionality : 

 - Wysiwig
 - publication text, photos gallery, videos, wiki...
 - keep historic of modification.
 - identification with OpenID
 - multilanguage

Internal :
 - parser url
 - sessions
 - global
 - sqlite3
 - svn 
 - Internal framework (module for publication objects).
 - viewer HTML
